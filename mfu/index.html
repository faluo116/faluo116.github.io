#3、文本框
示例：
<input type="text" class="inbox" text="这是一个文本框" />

备注：
灰色边框，扁平类型。字1.2em左移6px，宽300高30。
#4、按钮
示例：
<button class="btn" onclick="alert('I don\'t money!')">普通按钮</button>

<button class="btn_submit" onclick="alert('I don\'t money!')">提交按钮</button>

<button class="btn_warning" onclick="alert('I don\'t money!')">警示按钮</button>

备注：
按用途分为两种：普通按钮，提交(submit)按钮和红色的警示按钮。一般直接用即可，不需要改。
#5、类型块
示例：
<span class="type_tab" style="background-color:#0F00FF;">Movie</span>
Movie
<span class="type_tab" style="background-color:#FF0000;">TV</span>
TV
<span class="type_tab" style="background-color:#009900;">OVA</span>
OVA

备注：
用来突出标注类型。颜色需要另行指定。因为默认字体都是白色的，所以大多需要指定一个醒目的深色调。
#6、表格
示例：
<table>
    <tr>
        <th>测试字段一</th>
        <th>测试字段二</th>
        <th>测试字段三</th>
    </tr>
    <tr>
        <td>行1.1</td>
        <td>行1.2</td>
        <td>行1.3</td>
    </tr>
    <tr>
        <td>行2.1</td>
        <td>行2.2</td>
        <td>行2.3</td>
    </tr>
</table>
测试字段一 测试字段二 测试字段三
行1.1 行1.2 行1.3
行2.1 行2.2 行2.3
备注：
导入CSS后，直接写表格默认的语法即可。会生成一个灰色风格的表格。

#8、引用
示例：
<blockquote>这里是引用。</blockquote>

编辑距离（Edit Distance），又称Levenshtein距离，是指两个字串之间，由一个转成另一个所需的最少编辑操作次数。 许可的编辑操作包括将一个字符替换成另一个字符，插入一个字符，删除一个字符。 一般来说，编辑距离越小，两个串的相似度越大。

备注：
在原来的blockquote标签基础上加了左边框和底色。
