#1、标题
id选择器
示例：
<div id ="title">这是标题</div>
备注：
粗体，字号26。
#2、链接
类选择器
示例：
<a href="#" class="web">这是链接</a>
这是链接
备注：
没有指定字号。
#3、文本框
类选择器
示例：
<input type="text" class="inbox" text="这是一个文本框" />

备注：
灰色边框，扁平类型。字20px左移6px，宽300高30。
#4、按钮
类选择器
示例：
<button class="btn" onclick="alert('I don\'t money!')">普通按钮</button>

<button class="btn_submit" onclick="alert('I don\'t money!')">提交按钮</button>

<button class="btn_warning" onclick="alert('I don\'t money!')">警示按钮</button>

备注：
按用途分为三种：普通按钮，提交(submit)按钮和红色的警示按钮。一般直接用即可，不需要改。
#5、类型块
类选择器
示例：
<span class="type_tab" style="background-color:#0F00FF;">Movie</span>
Movie
备注：
用来突出标注类型。颜色需要另行指定。因为默认字体都是白色的，所以大多需要指定一个醒目的深色调。
#6、表格
默认示例，不需要指定选择器
示例：
<table>
    <tr>
        <th>测试字段一</th>
        <th>测试字段二</th>
        <th>测试字段三</th>
    </tr>
    <tr>
        <td>行1.1</td>
        <td>行1.2</td>
        <td>行1.3</td>
    </tr>
    <tr>
        <td>行2.1</td>
        <td>行2.2</td>
        <td>行2.3</td>
    </tr>
</table>
测试字段一 测试字段二 测试字段三
行1.1 行1.2 行1.3
行2.1 行2.2 行2.3
备注：
导入CSS后，直接写表格默认的语法即可。会生成一个灰色风格的表格。
#7、分割线
id选择器
示例：
<div id ="split"></div>
备注：
灰色，左右留有余白的分割线。
